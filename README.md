# snake-game
A classic arcade Sssnake game developed using the Python Pygame library.
## Features: 
### I) The game has 2 modes -
- Open Maze
- Closed Maze

You can select mode from the starting screen

### II) The game can save your high-score, permanently.

### III) Fun SFX !!!

Install & Enjoy :)
